# Customer Portal App

This is a simple React app ready for deployment on Heroku.

## Local Development

```bash
npm install
npm start
```

## Build for Production

```bash
npm run build
```

## Deploy to Heroku

1. Commit your code to git.
2. Create a Heroku app:
   ```bash
   heroku create
   ```
3. Push to Heroku:
   ```bash
   git push heroku main
   ```
